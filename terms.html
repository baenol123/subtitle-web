이용약관

시행일: 2026년 7월 7일

1. 서비스의 성격

자막공방(이하 "본 사이트")은 영상·오디오 파일에서 자막을 추출하고 번역하는 도구를 무료로 제공하는 웹사이트입니다. 본 사이트는 별도의 서버 없이 이용자의 브라우저에서 동작하며, 음성 인식과 번역은 이용자가 직접 발급한 제3자 서비스(Groq, Anthropic)의 API를 통해 이루어집니다.

2. 비용

본 사이트의 이용은 무료입니다. 다만 자막 추출과 번역 과정에서 이용자의 API 키로 호출되는 제3자 서비스의 요금은 이용자 본인이 부담합니다. API 사용량과 요금은 각 서비스의 콘솔에서 확인할 수 있으며, 본 사이트는 이용자의 API 사용으로 발생한 비용에 대해 책임지지 않습니다.

3. 이용자의 책임

  • 이용자는 자신이 적법하게 이용할 권리가 있는 파일에 대해서만 본 도구를 사용해야 합니다.
  • 타인의 저작물을 무단으로 복제·배포하는 행위, 관련 법령을 위반하는 용도의 사용은 금지됩니다.
  • 생성된 자막과 번역 결과물의 이용에 따른 책임은 이용자에게 있습니다.
  • API 키의 관리 책임은 이용자에게 있습니다. 공용 기기에서는 사용 후 브라우저 데이터를 삭제해 주세요.

4. 결과물의 품질

자막 추출과 번역은 AI 모델에 의해 자동으로 수행되며, 그 결과에는 오류나 부정확한 내용이 포함될 수 있습니다. 본 사이트는 결과물의 정확성·완전성을 보증하지 않으며, 중요한 용도로 사용하기 전에는 반드시 결과물을 직접 검수하시기 바랍니다.

5. 서비스의 변경 및 중단

본 사이트는 사전 고지 없이 기능을 추가·변경하거나 서비스를 중단할 수 있습니다. 또한 제3자 서비스(Groq, Anthropic)의 정책 변경이나 장애로 인해 일부 기능이 동작하지 않을 수 있습니다.

6. 면책

본 사이트는 무료로 제공되는 도구로서, 사이트 이용으로 인해 발생한 직접적·간접적 손해(데이터 손실, API 비용, 결과물 오류로 인한 손해 등)에 대해 법령이 허용하는 범위 내에서 책임을 지지 않습니다.

7. 지식재산권

본 사이트의 디자인과 코드에 대한 권리는 운영자에게 있습니다. 이용자가 업로드한 파일과 생성된 자막·번역 결과물에 대한 권리는 본 사이트가 주장하지 않습니다.

8. 약관의 변경

본 약관이 변경되는 경우 이 페이지를 통해 고지하며, 시행일을 함께 갱신합니다.
