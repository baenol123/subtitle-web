■ 자막공방 (subtitle-web) — 구조 및 운영 가이드

────────────────────────────────────────
1. 폴더 구조
────────────────────────────────────────
  index.html      메인 페이지 (자막 추출·번역 도구 + 소개/FAQ 콘텐츠)
  about.html      사이트 소개
  privacy.html    개인정보처리방침 (애드센스 필수)
  terms.html      이용약관
  _template.html  ★ 새 도구/페이지를 만들 때 복사해서 쓰는 템플릿
  style.css       공통 스타일 (파스텔 테마, 디자인 토큰 기반)
  app.js          자막 추출·번역 로직 (index.html 전용)
  vendor/         ffmpeg 라이브러리 — 배포 시 반드시 포함, 수정 금지
  robots.txt      검색엔진 크롤링 설정
  sitemap.xml     사이트맵 (페이지 추가 시 갱신)

────────────────────────────────────────
2. 새 기능(도구) 추가하는 법
────────────────────────────────────────
  1) _template.html을 복사해 새 파일 생성 (예: sync.html)
  2) 파일 안의 [바꾸기] 표시를 전부 수정
  3) 도구 로직은 별도 js 파일로 작성 (예: sync.js)
  4) 모든 html 파일의 내비게이션/푸터에 링크 추가
  5) sitemap.xml에 새 페이지 URL 추가

  · 색/모양을 바꾸고 싶으면 style.css 맨 위 "디자인 토큰"의 색상 변수만 수정
  · app.js가 쓰는 id/class(index.html의 도구 영역)는 이름을 바꾸면 안 됨

────────────────────────────────────────
