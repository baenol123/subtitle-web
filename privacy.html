개인정보처리방침

시행일: 2026년 7월 7일

자막공방(이하 "본 사이트")은 이용자의 개인정보를 소중히 여기며, 관련 법령을 준수합니다. 본 방침은 본 사이트가 어떤 정보를 다루는지, 그 정보가 어디로 전송되는지, 이용자가 무엇을 통제할 수 있는지를 설명합니다.

1. 수집하는 개인정보

본 사이트는 회원가입이 없으며, 이름·이메일·연락처 등 개인 식별 정보를 수집하지 않습니다. 별도의 서버를 운영하지 않는 정적 웹사이트로, 이용자가 업로드한 파일이나 입력한 내용을 저장하는 데이터베이스가 존재하지 않습니다.

2. 업로드 파일의 처리

  • 이용자가 업로드한 영상·오디오 파일은 본 사이트의 서버로 전송되지 않으며, 이용자의 브라우저 내에서만 처리됩니다.
  • 영상에서 분리된 오디오는 자막 추출을 위해 이용자의 API 키로 Groq, Inc.의 음성 인식 서비스에 전송됩니다.
  • 추출되거나 업로드된 자막 텍스트는 번역을 위해 이용자의 API 키로 Anthropic, PBC의 AI 서비스에 전송됩니다.
  • 전송된 데이터의 처리와 보관은 각 서비스의 개인정보처리방침을 따릅니다: Groq 개인정보처리방침 · Anthropic 개인정보처리방침

3. 로컬 저장소(localStorage)

편의를 위해 다음 정보가 이용자의 브라우저 로컬 저장소에 저장됩니다. 이 정보는 이용자의 기기를 벗어나지 않습니다.

  • 이용자가 입력한 API 키 (Groq, Anthropic)
  • 언어·모델 등 도구 설정, 번역 톤 지시, 용어집

브라우저 설정에서 사이트 데이터를 삭제하면 위 정보는 모두 제거됩니다. 공용 컴퓨터에서는 사용 후 삭제를 권장합니다.

4. 쿠키 및 광고

본 사이트는 Google AdSense를 통한 광고를 게재할 수 있습니다. 광고 게재와 관련하여 다음 사항을 알려드립니다.

  • Google을 포함한 제3자 광고 사업자는 쿠키를 사용하여 이용자의 본 사이트 및 다른 웹사이트 방문 기록을 바탕으로 광고를 게재할 수 있습니다.
  • Google은 광고 쿠키를 사용하여 이용자에게 맞춤형 광고를 제공합니다. 이용자는 Google 광고 설정에서 맞춤형 광고를 비활성화할 수 있습니다.
  • 제3자 광고 사업자의 쿠키 사용에 대한 자세한 내용은 Google 광고 정책을 참고해 주세요.

5. 접속 기록 및 분석

본 사이트는 현재 별도의 방문자 분석 도구를 사용하지 않습니다. 다만 사이트가 호스팅되는 플랫폼(예: Cloudflare Pages, GitHub Pages)이 서비스 운영을 위해 접속 로그(IP 주소, 접속 시각 등)를 자동으로 수집할 수 있으며, 이는 해당 플랫폼의 정책을 따릅니다. 추후 분석 도구를 도입할 경우 본 방침을 갱신하여 고지합니다.

6. 아동의 개인정보

본 사이트는 아동을 대상으로 하지 않으며, 아동의 개인정보를 의도적으로 수집하지 않습니다.

7. 방침의 변경

본 방침이 변경되는 경우 이 페이지를 통해 고지하며, 시행일을 함께 갱신합니다. 중요한 변경이 있는 경우 사이트 공지로 안내합니다.
